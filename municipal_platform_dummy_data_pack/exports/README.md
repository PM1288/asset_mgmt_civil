# Exports

This folder contains ready-to-use data profiles for the **current repository schema**.

Each profile is self-contained and includes:
- property create payloads
- license create/transition plans
- ward metadata
- user seed references
- dashboard metrics
- workflow events
- audit events
- sample text documents
- import manifest

## Profiles
- `developer_seed` — small / fast
- `hybrid_demo` — medium / demo-ready

## Import target
These files are designed for the current FastAPI application and its property/license/document workflow model.

See `../integration/README.md` for import steps.
